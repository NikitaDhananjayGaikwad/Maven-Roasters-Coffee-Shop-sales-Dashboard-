# Maven Roasters Coffee Shop Sales Dashboard(MS Excel)
An Interactive Analysis Dashboard of sales of a Coffee Shop Created in Microsoft Excel

# Project Objective
The coffee shop owner wants to know about the sales according to the different categories of products 

# This Project is created to understand  
1. Sales trended over time
2. Days of the week tend to be busiest and why
3. Times of day tend to be most popular and does the same trend hold across all locations
4. Products sold most and least often and  Which drive the most revenue for the business

# Data Set
There are single CSV file including the Pivot Table and Pivot Charts(dashboard)
<a href="https://github.com/NikitaDhananjayGaikwad/Maven-Roasters-Coffee-Shop-sales-Dashboard-/blob/main/Coffee%20Shop%20Sales1.xlsx">"Coffee Shop Data"</a>

# Data Wrangling
In Microsoft Excel the Basic process of data wrangling was done. Specialy with the date and time Columns,calculating the revenue and storing in new column and removing the duplicate and null values.
After cleaning the data using the Pivot table the query was resolved and using the Pivot Chat the dashboards were created
